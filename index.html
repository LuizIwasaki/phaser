Phaser

Estrutura de jogos HTML5 para desktop e dispositivos móveis.

O que é?

Phaser é uma estrutura de jogo 2D usada para fazer jogos HTML5 para desktop e dispositivos móveis. É um software livre e desenvolvido pela Photon Storm. Phaser usa um renderizador Canvas e WebGL internamente e pode alternar automaticamente entre eles com base no suporte do navegador.

Para que serve?

Phaser é utilizado para desenvolvimento de jogos da qual sua principal função é fazê-lo rodar nas mais diversas versões de navegadores tanto desktop quanto mobile.


Como funciona?

Como dito no tópico anterior, o Phaser veio para possibilitar um fácil desenvolvimento de jogos HTML5 para as mais variadas versões de navegador, tanto de celular quanto de computador.
